# Python-Caesar-Cipher

![Caesar_Cipher_Wheel](https://i.postimg.cc/bw9Q2Pyj/GH1.webp)

A [Caesar cipher](https://en.wikipedia.org/wiki/Caesar_cipher) is one of the simplest and most widely known [encryption](https://en.wikipedia.org/wiki/Encryption) techniques used in [cryptography](https://en.wikipedia.org/wiki/Cryptography). It is a type of [substitution cipher](https://en.wikipedia.org/wiki/Substitution_cipher) in which each letter in the [plaintext](https://en.wikipedia.org/wiki/Plaintext) is replaced by a letter some fixed number of positions along the [alphabet](https://en.wikipedia.org/wiki/Alphabet). For example, with a left shift of 3, D would be replaced by A, E would become B, and so on. The method is named after [Julius Caesar](https://en.wikipedia.org/wiki/Julius_Caesar), who used it in his private correspondence.
